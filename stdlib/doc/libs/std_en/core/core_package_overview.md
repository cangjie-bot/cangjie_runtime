# std.core

## Function Introduction

The core package is the standard library's core package, providing the most fundamental API capabilities for Cangjie language programming.

It provides built-in types (signed integers, unsigned integers, floating-point types, etc.), common functions (print, println, eprint, etc.), common interfaces (ToString, Hashable, Equatable, Collection, etc.), common classes and structures (Array, String, Range, etc.), and common exception classes (Error, Exception and some of their subclasses).

> **Note:**
> 
> The core package does not need to be explicitly imported, it is imported by default.
